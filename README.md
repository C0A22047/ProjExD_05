# ゲーム のタイトル
よけろ!!!
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
ジャンプや上下左右にプレイヤーのボールを動かして、線と球を避ける。

## ゲームの実装
###共通基本機能
* 基本行動に関するクラス
* 線とドットの親クラス
* 垂直の線に関するクラス
* 水平の線に関するクラス
* ドットに関するクラス。
### 担当追加機能
* 一瞬停止機能
* タイム機能
